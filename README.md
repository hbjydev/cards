## Running

Cards is very easy to install. The recommended way is
[with Docker](#with-docker), but you can also run
[by building](#by-building).

### With Docker

```shell
$ docker run -d -p 3000:3000 --name cards itshaydendev/cards
```

### By Building
```shell
$ make
$ build/cards
```
